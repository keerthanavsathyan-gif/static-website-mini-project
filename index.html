Cloud Computing Mini Project

Deploying a Static Website Using Cloud Hosting

About This Project

This project demonstrates how a static website can be deployed using a cloud-based hosting platform. Static websites contain fixed content written using HTML and CSS and do not require server-side processing.

What is Cloud Computing?

Cloud computing is the delivery of computing services such as storage, servers, and networking over the internet. It allows users to access resources without managing physical hardware.

Features of Cloud Hosting

  • High availability and reliability
  • Scalable and flexible resources
  • Cost-effective and easy to maintain
  • Accessible from anywhere via the internet

Technologies Used

  • HTML for structure
  • CSS for styling
  • GitHub Pages for cloud hosting

Conclusion

This mini project helped in understanding the concept of cloud deployment and static website hosting. It shows how cloud platforms can be used to deploy applications efficiently and securely.
